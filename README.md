# DeclinationNames

Small project for declination names to numbers on russian language. Example:

### call
``get(localizeNumber: i, one: "Один", format: ("час", "часа", "часов")) + " назад"``

``0 часов назад, 
Один час назад,

8 часов назад, 
9 часов назад, 
10 часов назад,
